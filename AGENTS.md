# AGENTS.md — Marble Cake Family

## Contexto permanente

Este es un proyecto real para la página pública de Marble Cake Family, no una maqueta desechable. La base debe crecer hacia historia, conexión con Dummeez, merch, drops, galería, updates y contacto.

## Reglas de trabajo

- Responder y documentar en español mexicano neutro, salvo que el contenido público del sitio requiera inglés.
- No usar copy de relleno ni decir que el proyecto es demo.
- Mantener la conexión con Dummeez visible como parte del universo Marble Cake.
- Extender la estructura existente antes de meter frameworks, wrappers o rediseños grandes.
- Priorizar composición, spacing, tipografía y claridad antes que agregar cajas, sombras o capas innecesarias.
- Validar con `npm run build` antes de cerrar cambios.

## Dirección visual inicial

- Fondo base: `public/assets/marble-cake.png`.
- Estética: editorial, cálida, oscura, con acentos tipo caramelo/mármol.
- Evitar look genérico de landing SaaS, gradientes neón, emojis y placeholders falsos.
